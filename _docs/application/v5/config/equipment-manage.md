Use the Table to search, sort, and manage your agency's saved equipment records.

## Table Navigation

- **Sorting**: Click any column header to sort the table by that field (e.g., Code or Model).
- **Pagination**: Use the navigation arrows or page numbers beneath the table to move through your records. You can adjust the number of rows displayed by using the **Items/page** dropdown.
- **Search**: Use the search box on the right to filter the table by keyword. This search updates the table in real-time. Click the **X** button to clear your search and return to the full list.

## Action Buttons

Located in the far right column of each row:

- ![Edit Icon]({{ site.baseurl }}/assets/images/docs/action-button-edit.png) **Edit (Pencil)**: Opens the original equipment form with all fields populated for modification.
- ![Delete Icon]({{ site.baseurl }}/assets/images/docs/action-button-delete.png) **Delete (Red Trash Can)**: Deletes the selected record from the system.
